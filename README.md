## Getting Started
Follow these steps to get started with the Parachain Node.
***
### Rust Setup
`curl https://sh.rustup.rs -sSf | sh`
***
### Clone Project From Github
```buildoutcfg
git clone https://github.com/listenofficial/listen-parachain.git
```
### Build
```buildoutcfg
cd listen-parachain

cargo build --release
```
